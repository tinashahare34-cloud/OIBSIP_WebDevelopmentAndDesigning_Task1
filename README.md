# OIBSIP_WebDevelopment_Task1
## Simple Calculator Web Application

This project is a Simple Calculator built using HTML, CSS, and JavaScript as part of the Oasis Infobyte (OIBSIP) Web Development Internship. The calculator performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Objective
The objective of this task is to develop a simple calculator that allows users to perform basic mathematical operations through an interactive web interface.

## Tools & Technologies Used
- HTML
- CSS
- JavaScript
- Visual Studio Code
- Web Browser (Google Chrome)

## Project Files
- index.html – Structure of the calculator
- style.css – Styling of the calculator
- script.js – JavaScript logic for calculator operations

## Features
- Addition, Subtraction, Multiplication, Division
- Clear button (C)
- Delete button (DEL)
- Result calculation (=)
- Keyboard input support
- Simple and user-friendly interface

## Outcome
A fully functional calculator web application was successfully created using HTML, CSS, and JavaScript. The project demonstrates basic web development skills and JavaScript event handling.
